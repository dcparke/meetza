# Meetza Privacy Policy

Last updated: 07/17/2026

Meetza is an open-source browser extension that helps users generate
Google Calendar availability while composing messages in Gmail.

## Information Meetza handles

Meetza may process:

- Google Calendar FreeBusy availability
- Calendar identifiers and email addresses entered by the user
- Google authentication information required to access the Calendar
  FreeBusy API
- Locally saved extension preferences, such as meeting duration,
  workday hours, time zone, break duration, and whether Gmail
  integration is enabled
- Recent calendar email addresses, only when the user enables the
  optional "Remember Recent People" setting

Meetza does not read or collect Gmail message bodies, subjects,
recipients, or attachments.

## How information is used

Information is used only to provide Meetza's calendar-availability
functionality and user-selected extension settings.

Calendar availability is requested directly from the Google Calendar
FreeBusy API after the user authorizes access through Google OAuth.

## Local storage

Meetza stores its settings locally using browser extension storage.

When "Remember Recent People" is enabled, recently entered calendar
email addresses may also be stored locally. This feature is disabled
by default.

## Data transmission and sharing

Meetza has no developer-operated backend server.

User data is not transmitted to the Meetza developer, sold, rented,
used for advertising, used for analytics, or shared for marketing.

Authentication and Calendar FreeBusy requests are transmitted directly
between the extension and Google's services using HTTPS as necessary
to provide the extension's functionality.

## Remote code

Meetza does not download or execute remotely hosted code. All
executable code is included in the installed extension package.
